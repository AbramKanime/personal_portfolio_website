Quiz App

A quizzical that tests the player on their knowledge of general questions ranging from sports, science, history, movies, and general knowledge.

Live Link

Project Overview

The app is designed to fetch questions from an Api for every round of game played, and display the questions and answer options randomly. The player selects the likely answers to the questions, and clicks the "Check answers" button, which then shows the result by coloring the answers they got correct/wrong while coloring the right answers. The player can replay the game multiply times. The app was built primarily using React JS.

The app was built and designed to be highly responsive; hence it can adjust to any screen size.
